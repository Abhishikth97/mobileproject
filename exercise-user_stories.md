# exercise user story
## User Stories

### Signup
- As a new user, I want to register with username, email, and password so that I can create an account.

### Login
- As a registered user, I want to log in with email and password so that I can access my account.

### Home Screen
- As a user, I want to view an overview of my data on the home screen so that I can monitor my progress.

### Detail Screen
- As a user, I want detailed information about an item so that I can make informed decisions.
